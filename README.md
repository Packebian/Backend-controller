# Backend

a [Sails](http://sailsjs.org) application


## Prepare environment
Install dependencies and update the PATH
```sh
$ npm install
$ source env.sh
```

## Build & development
The application is built and served with npm
```sh
$ npm start
```

## Testing
The tests are run with npm
```sh
$ npm test
```

The modules used for testing are
- mocha
- supertest
- chai
